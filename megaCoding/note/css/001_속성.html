CSS 기본속성

색, 배경, 텍스트, 글꼴, 링크, 리스트, 테이블에 대한 기본 속성을 살펴봅니다.


COLOR

CSS에서 색을 표현하는 방법에는 다음과 같이 세 가지 방법이 있습니다.

  1. 색상 이름으로 표현

    2. .blue { color:blue; }

    현재는 대부분의 브라우저가 140개의 색상 이름을 지원하고 있습니다.

  2. RGB 색상값으로 표현

    3. .blue { color:rgb(0,0,255); }

    모니터나 스크린은 빨간색(Red), 녹색(Green), 파란색(Blue)을 혼합하여 색을 표현합니다.

    따라서 HTML에서도 이 세 가지 색을 가지고 색을 표현하는 RGB 색상을 사용합니다.

    RGB 색상의 기본색(Red, Green, Blue)은 각각 0부터 255까지의 범위를 가집니다.

  3. 16진수 색상값으로 표현

    4. .blue { color:#0000FF; }

    .green { color: #008000; }

    .silver { color: #C0C0C0; }

    16진수 색상값으로 표현

    16진수 색상값은 RGB 색상값을 각각 16진수로 변환한 것입니다.

    따라서 RGB 색상의 기본색(Red, Green, Blue)은 각각 00부터 FF까지의 범위를 가집니다.

배경

CSS에서 사용할 수 있는 background 속성은 다음과 같습니다.

  1. background-color

    2. background-color 속성은 해당 HTML 요소의 배경색(background color)을 설정합니다.

    body { background-color: lightblue; }

  2. background-image

    3. 매소드 { background-image: url(이미지파일의 위치); }

    body { background-image: url("/images/img_background.png"); }

  3. background-repeat

    4. 배경 이미지를 수평이나 수직 방향으로만 반복되도록 설정할 수 있습니다.

    수평반복 body { background-image: url("/images/img_background.png"); background-repeat: repeat-x;}

    수직반복 body { background-image: url("/images/img_background.png"); background-repeat: repeat-y;}

    배경 이미지가 반복되지 않고 한 번만 나타나게 할 수도 있습니다.

    반복없음 body { background-image: url("/images/img_background.png"); background-repeat: no-repeat;}

  4. background-position

    5. background-position 속성은 반복되지 않는 배경 이미지의 상대 위치(relative position)를 설정합니다.

    body {

    background-image: url("/images/img_background.png");

    background-repeat: no-repeat;}

    background-position: top right;

    }

  5. background-repeat

    6. background-attachment 속성을 사용하여 위치가 설정된 배경 이미지를 해당 위치에 고정시킬 수도 있습니다.

    이렇게 고정된 배경 이미지는 스크롤과는 무관하게 화면의 위치에서 이동하지 않습니다.

    body {

    background-image: url("/images/img_background.png");

    background-repeat: no-repeat;}

    background-position: top right;

    background-attachment: fixed;

    }

  • background 속성 한 번에 적용하기

body { background-image: url("/images/img_background.png") no-repeat top right fixed; }

text

CSS에서 사용할 수 있는 대표적인 text 속성은 다음과 같습니다.

  1. color : 텍스트의 색상을 설정
  2. direction : 텍스트가 써지는 방향을 설정
  3. letter-spacing : 텍스트 내에서 글자 사이의 간격을 설정
  4. word-spacing :텍스트 내에서 단어 사이의 간격을 설정
  5. text-indent : 단락의 첫 줄에 들여쓰기할지 안 할지를 설정
  6. text-align :텍스트의 수평 방향 정렬을 설정
  7. text-decoration :텍스트에 여러 가지 효과를 설정하거나 제거

    8. tip! text-decoration 속성값을 none으로 설정하여 링크(link)가 설정된 텍스트의 밑줄을 제거하는데 자주 사용합니다.

  8. text-transform : 텍스트에 포함된 영문자에 대한 대소문자를 설정
  9. line-height : 텍스트의 줄 간격을 설정
  10. text-shadow : 텍스트에 그림자 효과를 설정

font

CSS에서 사용할 수 있는 font 속성은 다음과 같습니다.

  1. font-family

    2. font-family 속성은 하나의 글꼴만을 설정할 수도 있고, 여러 개의 글꼴을 같이 설정할 수도 있습니다.

    font-family 속성값이 여러 개의 글꼴로 설정되어 있으면, 웹 브라우저는 위에서부터 순서대로 글꼴을 읽어 들입니다.

  2. font-style

    3. font-style 속성은 주로 이탤릭체를 사용하기 위해 사용하며, 다음과 같이 3가지의 속성값을 가집니다.

    .normal { font-style: normal; } 텍스트에 어떠한 스타일도 적용하지 않습니다.

    .italic { font-style: italic; } 텍스트를 이탤릭체로 나타냅니다.

    .oblique { font-style: oblique; } 텍스트를 기울입니다. 이 속성값은 italic과 매우 유사하지만 지원하는 웹 브라우저가 거의 없습니다.

  3. font-variant : 텍스트에 포함된 영문자 중 모든 소문자를 작은 대문자(small-caps) 글꼴로 변경
  4. font-weight : 텍스트를 얼마나 두껍게 표현할지를 설정
  5. font-size : 텍스트의 크기를 설정

    6. 절대적 크기 : 픽셀(pixel)

    상대적 크기 : 백분율 단위(%)는 기본 크기를 100%로 놓고, 그에 대한 상대적인 크기를 설정

    상대적 크기 : 배수 단위(em)는 해당 글꼴(font)의 기본 크기를 1em으로 놓고, 그에 대한 상대적인 크기를 설정

link

링크(link)에는 color, font-family, background 속성 등 CSS의 다양한 속성들을 적용할 수 있습니다.

  1. link : 링크의 기본 상태이며, 사용자가 아직 한 번도 해당 링크를 통해 연결된 페이지를 방문하지 않은 상태
  2. visited : 사용자가 한 번이라도 해당 링크를 통해 연결된 페이지를 방문한 상태
  3. hover : 사용자의 마우스 커서가 링크 위에 올라가 있는 상태
  4. active : 사용자가 마우스로 링크를 클릭하고 있는 상태
  5. focus : 키보드나 마우스의 이벤트(event) 또는 다른 형태로 해당 요소가 포커스(focus)를 가지고 있는 상태

list

CSS에서 사용할 수 있는 list-style 속성은 다음과 같습니다.

  1. list-style-type : 리스트에 다양한 마커(marker)를 적용할 수 있습니다.

    2. .square { list-style-type: square; }

  2. list-style-image : 마커(marker)로 자신만의 이미지를 사용할 수 있습니다.

    3. .imageMarker { list-style-image: url("/images/img_list_marker.png"); }

  3. ist-style-position : 리스트 요소의 위치를 설정할 수 있습니다.

    4. .inside { list-style-position: inside; }

  • list-style 속성 한 번에 적용하기

ul { list-style: square inside url("images/img_list_marker.png"); }

table css

테이블에 다음 속성을 사용하여 CSS 스타일을 적용할 수 있습니다.

  1. border : 테이블의 테두리(border)를 설정
  2. border-collapse : 해당 테이블 의 테두리를 합침
  3. border-spacing : 테이블 요소(th, td)간의 여백을 설정
  4. caption-side : th, td 내부에서 텍스트의 수평 방향 정렬을 설정
  5. empty-cells : th, td 내부에서 텍스트의 수직 방향 정렬을 설정
  6. table-layout : 테이블에 사용되는 레이아웃 알고리즘을 설정
