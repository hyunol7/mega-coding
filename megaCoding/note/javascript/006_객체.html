객체(object)란?


자바스크립트에서 객체(Object)는 데이터를 효과적으로 구조화하고, 관리하기 위한 중요한 요소입니다. 객체는 이름과 값의 쌍으로 구성된 프로퍼티(Property)를 포함하며, 이러한 프로퍼티들의 집합체입니다. 객체는 중괄호 {}로 표현되며, 프로퍼티들은 쉼표로 구분되어 나열됩니다.
자바스크립트에서 객체를 생성하는 방법은 여러 가지가 있습니다. 가장 일반적인 방법은 객체 리터럴을 사용하는 것입니다.
// 빈 객체 생성
var person = {};

// 프로퍼티 추가
person.name = "John";
person.age = 30;
person.gender = "male";

// 또는 프로퍼티를 초기화하는 단계에서 선언
var person = {
    name: "John",
    age: 30,
    gender: "male"
};

위의 코드에서 person은 객체이며, name, age, gender는 해당 객체의 프로퍼티들입니다. 이러한 프로퍼티는 각각 이름과 값으로 식별됩니다.
또한, 객체는 메서드(Method)라 불리는 함수도 포함할 수 있습니다. 이 함수는 해당 객체에서 수행할 수 있는 동작을 정의합니다.
var person = {
    name: "John",
    age: 30,
    gender: "male",
    sayHello: function() {
    console.log("Hello, my name is " + this.name);
    }
};

// 메서드 호출


person.sayHello(); // 출력: Hello, my name is John

여기서 sayHello는 person 객체의 메서드로서, 객체의 상태나 프로퍼티에 접근할 수 있습니다. this 키워드는 현재 객체를 참조합니다.
자바스크립트에서는 다양한 방식으로 객체를 생성하고 다룰 수 있으며, 객체는 자바스크립트의 핵심 요소 중 하나로 프로그래밍에서 광범위하게 사용됩니다.

상속(inheritance)


상속(inheritance)이란 새로운 클래스에서 기존 클래스의 모든 프로퍼티와 메소드를 사용할 수 있는 것을 의미합니다.

프로토타입(prototype)


자바스크립트의 모든 객체는 프로토타입(prototype)이라는 객체를 가지고 있습니다.
모든 객체는 그들의 프로토타입으로부터 프로퍼티와 메소드를 상속받습니다.
이처럼 자바스크립트의 모든 객체는 최소한 하나 이상의 다른 객체로부터 상속을 받으며, 이때 상속되는 정보를 제공하는 객체를 프로토타입(prototype)이라고 합니다.
