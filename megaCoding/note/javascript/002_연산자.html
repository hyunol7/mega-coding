연산자(operator)


자바스크립트는 여러 종류의 연산을 위한 다양한 연산자(operator)를 제공하고 있습니다.

자바스크립트에서 연산자는 값을 계산하고 조작하는 데 사용되는 기호 또는 키워드입니다. 다양한 종류의 연산자가 있으며, 주로 산술 연산, 비교 연산, 논리 연산 등으로 구분됩니다

1. 산술 연산자산술 연산자는 숫자 간의 계산에 사용됩니다.


산술 연산자 설명

+ 왼쪽 피연산자의 값에 오른쪽 피연산자의 값을 더함.
- 왼쪽 피연산자의 값에서 오른쪽 피연산자의 값을 뺌.
* 왼쪽 피연산자의 값에 오른쪽 피연산자의 값을 곱함.
/ 왼쪽 피연산자의 값을 오른쪽 피연산자의 값으로 나눔.
% 왼쪽 피연산자의 값을 오른쪽 피연산자의 값으로 나눈 후, 그 나머지를 반환함.

+ (더하기): 두 값을 더합니다.
let sum = 5 + 3; // 8

- (빼기): 오른쪽 값을 왼쪽 값에서 뺍니다.
let difference = 10 - 4; // 6

* (곱하기): 두 값을 곱합니다.
let product = 3 * 7; // 21

/ (나누기): 왼쪽 값을 오른쪽 값으로 나눕니다.
let quotient = 15 / 3; // 5

% (나머지): 왼쪽 값을 오른쪽 값으로 나눈 후 나머지를 반환합니다.
let remainder = 10 % 3; // 1

2. 비교 연산자


비교 연산자는 값을 비교하여 불리언 값을 반환합니다

== (동등): 값이 동등하면 true, 아니면 false.
console.log(5 == '5'); // true (값만 비교)

=== (일치): 값과 데이터 타입이 모두 동등하면 true, 아니면 false.
console.log(5 === '5'); // false (값과 데이터 타입 비교)

!= (부등): 값이 다르면 true, 같으면 false.
console.log(10 != '10'); // false

!== (불일치): 값이나 데이터 타입 중 하나라도 다르면 true, 둘 다 같으면 false.
console.log(10 !== '10'); // true
> (크다), < (작다), >= (크거나 같다), <= (작거나 같다): 숫자 간의 크기를 비교합니다.

3. 논리 연산자


논리 연산자는 불리언 값 간의 논리 연산에 사용됩니다.

&& (논리 AND): 양쪽 모두 true일 때만 true.
console.log(true && false); // false

|| (논리 OR): 양쪽 중 하나라도 true이면 true.
console.log(true || false); // true

! (논리 NOT): 불리언 값을 반전시킵니다.
console.log(!true); // false

4. 할당 연산자


할당 연산자는 변수에 값을 할당합니다.
= (할당): 오른쪽 값을 왼쪽 변수에 할당합니다.

let x = 10;
+=, -=, *=, /= 등 (복합 할당): 현재 값과 연산하여 결과를 변수에 할당합니다.

let y = 5;
y += 3; // y = y + 3;
자바스크립트에는 이외에도 다양한 연산자가 있으며, 이를 활용하여 복잡한 계산과 조작을 수행할 수 있습니다.

5. 증감 연산자


증감 연산자는 변수의 값을 1씩 증가시키거나 감소시킵니다.

++ (증가): 변수의 값을 1 증가시킵니다.
let count = 0;
count++; // count는 이제 1

-- (감소): 변수의 값을 1 감소시킵니다.
let total = 10;
total--; // total은 이제 9

6. 조건 (삼항) 연산자


삼항 연산자는 조건에 따라 값을 선택합니다.

조건 ? 값1 : 값2 (삼항 연산자): 조건이 true이면 값1을 반환하고, false이면 값2를 반환합니다.
let age = 20;
let result = age >= 18 ? '성인' : '미성년자';

7. 비트 연산자


비트 연산자는 이진수로 표현된 숫자의 비트를 조작합니다.
& (비트 AND), | (비트 OR), ^ (비트 XOR): 비트별 AND, OR, XOR 연산을 수행합니다.
let a = 5; // 이진수: 0101
let b = 3; // 이진수: 0011
console.log(a & b); // 0101 & 0011 = 0001 (2진수), 1 (10진수)
이러한 연산자들을 조합하여 다양한 작업을 수행할 수 있습니다. 자바스크립트에서 연산자는 프로그래밍에서 핵심적인 역할을 하며, 변수와 함께 사용하여 다양한 로직을 구현할 수 있습니다.

8.쉼표 연산자


쉼표 연산자를 for 문에서 사용하면, 루프마다 여러 변수를 동시에 갱신할 수 있습니다
// 루프마다 i의 값은 1씩 증가하고, 동시에 j의 값은 1씩 감소함.
for (var i = 0, j = 9; i <= j; i++, j--) {
document.write("i의 값은 " + i + "이고, j의 값은 " + j + "입니다.<br>");
}

9.delete 연산자


delete 연산자는 피연산자인 객체, 객체의 프로퍼티(property) 또는 배열의 요소(element) 등을 삭제해 줍니다.
피연산자가 성공적으로 삭제되었을 경우에는 참(true)을 반환하고, 삭제하지 못했을 경우에는 거짓(false)을 반환합니다.
이 연산자는 피연산자가 단 하나뿐인 단항 연산자이며, 피연산자의 결합 방향은 오른쪽에서 왼쪽입니다.
var arr = [1, 2, 3]; // 배열 생성
delete arr[2]; // 배열의 원소 중 인덱스가 2인 요소를 삭제함.
document.write(arr + "<br>"); // [1, 2, ]
// 배열에 빈자리가 생긴 것으로 undefined 값으로 직접 설정된 것은 아님.
document.write(arr[2] + "<br>");
// 배열의 요소를 삭제하는 것이지 배열의 길이까지 줄이는 것은 아님.
document.write(arr.length);

10.typeof 연산자


typeof 연산자는 피연산자의 타입을 반환합니다.
이 연산자는 피연산자가 단 하나뿐인 단항 연산자이며, 피연산자의 결합 방향은 오른쪽에서 왼쪽입니다.
자바스크립트에서 많이 사용하는 값과 그 값에 대한 typeof 연산자의 결괏값은 다음과 같습니다.

값 typeof 연산자의 결괏값
숫자, NaN number

문자열

 string
true, false boolean
null object
undefined undefined
함수 function
함수가 아닌 객체 object

 

 

 

 

 

 

 

 

11.instanceof 연산자는 피연산자인 객체가 특정 객체의 인스턴스인지 아닌지를 확인해 줍니다.


피연산자가 특정 객체의 인스턴스이면 참(true)을 반환하고, 특정 객체의 인스턴스가 아니면 거짓(false)을 반환합니다.
이 연산자는 두 개의 피연산자를 가지는 이항 연산자이며, 피연산자들의 결합 방향은 왼쪽에서 오른쪽입니다.
var str = new String("이것은 문자열입니다.");
str instanceof Object; // true
str instanceof String; // true
str instanceof Array; // false
str instanceof Number; // false
str instanceof Boolean; // false

인스턴스와 객체에 대한 더 자세한 사항은 자바스크립트 객체의 개념 수업에서 확인할 수 있습니다.
void 연산자
void 연산자는 피연산자로 어떤 타입의 값이 오던지 상관없이 언제나 undefined 값만을 반환합니다.

이 연산자는 피연산자가 단 하나뿐인 단항 연산자이며, 피연산자의 결합 방향은 오른쪽에서 왼쪽입니다.
<a href="javascript:void(0)">이 링크는 동작하지 않습니다.</a>
<a href="javascript:void(document.body.style.backgroundColor='yellow')">
이 링크도 동작하지 않지만, HTML 문서의 배경색을 바꿔줍니다.
</a>
위의 예제처럼 void 연산자는 정의되지 않은 원시 타입의 값을 얻기 위해 void(0)과 같은 형태로 종종 사용됩니다.
