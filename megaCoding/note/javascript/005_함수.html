함수(function)란?


함수(function)란 하나의 특별한 목적의 작업을 수행하도록 설계된 독립적인 블록을 의미합니다.

함수 선언


함수를 선언할 때는 function 키워드를 사용합니다.
function greet(name) {
    console.log("안녕하세요, " + name + "님!");
}

greet("John");
위의 예제에서 greet는 함수의 이름이고, name은 매개변수(parameter)입니다. 함수를 호출할 때 greet("John")와 같이 인수(arguments)를 전달할 수 있습니다.

함수 표현식


함수 표현식은 변수에 함수를 할당하는 방식으로 함수를 생성합니다.
let multiply = function (a, b) {
    return a * b;
};
let result = multiply(3, 4);
console.log(result); // 12

화살표 함수 (Arrow Function)


화살표 함수는 ES6에서 도입된 새로운 함수 선언 방식입니다.
let add = (x, y) => x + y;
console.log(add(2, 3)); // 5
화살표 함수는 간결하게 함수를 표현할 수 있으며, 주로 콜백 함수나 간단한 함수를 정의할 때 사용됩니다.

함수의 반환값


함수는 값을 반환할 수 있습니다. return 키워드를 사용하여 함수 내에서 값을 반환하고, 호출한 곳에서 그 값을 사용할 수 있습니다.
function add(a, b) {
    return a + b;
}
let sum = add(3, 4);
console.log(sum); // 7

함수 스코프


자바스크립트에서는 함수 스코프가 적용되어 함수 내에서 선언된 변수는 해당 함수 내에서만 유효합니다.
function example() {
let localVar = "함수 내에서만 유효한 변수";
    console.log(localVar);
}
example();
console.log(localVar); // 오류: localVar는 정의되지 않음
함수는 코드의 모듈화와 재사용성을 높이는 데에 중요한 역할을 합니다.

매개변수와 인수


매개변수는 함수에 전달되는 값을 받는 변수를 의미하며, 인수는 함수를 호출할 때 전달되는 값입니다.

function greet(name) {
    console.log("안녕하세요, " + name + "님!");
}

greet("Jane");
위의 예제에서 name은 매개변수이고, "Jane"은 인수입니다.

기본 매개변수(Default Parameters)


ES6에서는 함수에 기본 매개변수를 설정할 수 있습니다.

function multiply(a, b = 1) {
    return a * b;
}

console.log(multiply(5)); // 5
console.log(multiply(5, 3)); // 15
b에 기본값으로 1이 설정되어 있으므로, multiply(5)와 같이 하나의 인수만 전달해도 기본값이 사용됩니다.

Rest 매개변수


Rest 매개변수는 함수에 여러 개의 인수를 배열로 받을 수 있게 합니다.
function sum(...numbers) {
    return numbers.reduce((total, num) => total + num, 0);
}
console.log(sum(1, 2, 3, 4, 5)); // 15

즉시 실행 함수(IIFE, Immediately Invoked Function Expression)


즉시 실행 함수는 선언과 동시에 즉시 실행되는 함수입니다.
(function() {
console.log("즉시 실행 함수");
})();

클로저 (Closure)


클로저는 함수와 함수가 선언된 어휘적 환경(Lexical Environment)의 조합입니다. 함수 내부에서 외부 변수를 참조할 수 있게 합니다.
function outer() {
     let outerVar = "외부 변수";
     function inner() {
          console.log(outerVar);
     }
    return inner;
}
let closureFunc = outer();
closureFunc(); // "외부 변수"

함수 호이스팅 (Function Hoisting)


자바스크립트에서 함수 선언문은 코드의 최상단으로 끌어올려지는 현상을 호이스팅이라고 합니다.
sayHello(); // "안녕하세요!"
function sayHello() {
     console.log("안녕하세요!");
}

이렇게 함수 선언문은 선언되기 전에 호출할 수 있습니다.
함수는 프로그래밍에서 중요한 개념이며, 이를 통해 코드를 모듈화하고 재사용성을 높일 수 있습니다.
