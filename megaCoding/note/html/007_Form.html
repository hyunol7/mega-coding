Form 요소

웹 페이지에서는 form 요소를 사용하여 사용자로부터 입력을 받을 수 있다.
또한, 사용자가 입력한 데이터를 서버로 보낼때도 form 요소를 사용한다

method 속성

method 속성을 통해 명시할 수 있는 form 요소의 전달 방식은 GET 방식과 POST 방식으로 나눠집니다.

  • GET 방식은 주소에 데이터(data)를 추가하여 전달하는 방식입니다.
  • POST 방식은 데이터(data)를 별도로 첨부하여 전달하는 방식입니다.

HTML의 input 요소

텍스트 입력

input태그의 type 속성값을 "text"로 설정하면, 사용자로부터 한 줄의 텍스트를 입력받을 수 있습니다.

비밀번호 입력

input태그의 type 속성값을 "password"로 설정하면, 사용자로부터 비밀번호를 입력받을 수 있습니다.

라디오 버튼

input 태그의 type 속성값을 "radio"로 설정하면, 사용자로부터 여러 개의 옵션(option) 중에서 단 하나의 옵션만을 입력받을 수 있습니다.

체크박스

input태그의 type 속성값을 "checkbox"로 설정하면, 사용자로부터 여러 개의 옵션 중에서 다수의 옵션을 입력받을 수 있습니다.
체크박스는 라디오 버튼과는 달리 여러 개의 옵션을 한 번에 입력받을 수 있습니다.

파일 선택

input태그의 type 속성값을 "file"로 설정하면, 사용자로부터 파일을 전송받을 수 있습니다.

파일 선택 박스

선택 입력

select 요소는 여러 개의 옵션이 드롭다운 리스트(drop-down-list)로 되어 있으며,
그 중, 단 하나의 옵션만을 입력받을 수 있습니다.

문장 입력

texare 요소는 사용자로부터 여러 줄의 텍스트를 입력받을 수 있습니다.

문장 입력

버튼

button 요소는 사용자로부터 여러 줄의 텍스트를 입력 받을 수 있습니다.

버튼

button 요소는 사용자가 누를 수 있는 버튼을 나타냅니다.

전송 버튼

input 태그의 type 속성값을 submit으로 설정하면, 사용자로부터 입력받은 데이터(data)를
서버의 폼 핸들러로 제출하는 버튼을 만들 수 있습니다.

필드셋(fieldset)
