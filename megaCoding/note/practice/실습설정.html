실습 개요

수업 시간에 배운 내용을 응용하여 실제 코딩을 진행할 수 있는 웹 페이지를 구현한 페이지 입니다.
실습 페이지는 수업에서 배운 내용을 바탕으로 프로그램를 구현하고, 프로그래밍 언어와 기술을 활용하여 내용을 확장하고 있습니다.
이를 통해 수업에서 배운 지식이 실제 프로젝트에 어떻게 적용되는지를 보여주고 있으며, 이를 통해 더 깊이 있는 학습과 응용력 향상시키 것 입니다.


강민정

숫자게임 프로그램
자바스크립트 강의에서 배운 if문과 랜덤 함수의 활용을 기반으로 하여, 자바스크립트를 활용한 숫자 맞추기 게임을 구현 했습니다. 이 실습으로 수업에서 배운 제어문과 함수를 적극적으로 활용하여 조건문을 이해했으며, 사용자가 입력한 값과 정답을 비교하고 그에 따른 이론적인 개념을 실전 하여 어떻게 응용할 수 있는지 코드를 통한 문제 해결 능력을 향상했습니다.

구구단 프로그램
수업에서 배운 for문을 활용하여 구구단 프로그램을 작성한 퀴즈를 응용해서 이번에는 함수를 활용하여 램덤으로 구구단을 맞추는 퀴즈 게임을 만들었습니다. 퀴즈에 하였던 for문과 다르게 함수를 사용하지 함수의 대한 개념을 이해된것 같습니다.

연산자 구현 프로그램
퀴즈 문제중 연산자 문제를 응용하여, 두 개의 숫자를 입력받아 모든 연산을 수행하여 출력 하는 프로그램을 개발했습니다. 사용자에게는 덧셈, 뺄셈, 곱셈, 나눗셈, 나머지 연산 값을 모두 수행하여 결과값을 출력하는 기능 기능을 구현했습니다.이를 통해 프로그래밍에서의 조건문과 연산자 활용에 대한 이해를 높아졌습니다.


박낙주

로또 프로그램
바스크립트 프로그램을 개발하였습니다.프로그램은 사용자로부터 입력받은 로또 구매 회수에 따라 지정된 횟수만큼 로또 번호를 생성하고, 각 회차별로 당첨 여부를 for문, switch문, while문과 같은 반복문을 적절히 활용하였고, 결과를 사용자에게 효과적으로 전달하기 위해 document 객체를 이용하여 DOM 조작을 수행했습니다.반복문과 조건문의 활용, DOM 조작 등 다양한 자바스크립트 개념을 효과적으로 활용하 였습니다.

위틀(WETL)
위틀(단어 추측 게임) 게임을 자바스크립트로 구현하였습니다. 사용자가 입력한 번로또 당첨 여부를 확인하는 자호와 정답 문자를 비교하여, 문자 위치에 따라 다른 스타일로 표시하는 기능을 구현했습니다. 프로그램은 사용자의 입력에 대한 결과에 따라 틀린 경우 빨간색, 숫자는 맞지만 위치가 틀린 경우 노란색, 그리고 정확히 일치하는 경우 녹색으로 input 박스의 스타일을 변환하는 기능을 구현하였습니다.

정육면체
CSS를 이용하여 정육면체를 구현하는 과정에서 3D 개념과 에니메이션 효과에 대한 이해를 높일 수 있었습니다.

최주현

가위바위보 프로그램
수업 시간에 배운 조건문을 활용하여, 자바스크립트로 가위바위보 게임을 구현했습니다. 사용자와 컴퓨터 간의 상호작용을 도입하여 onclick 이벤트를 사용하여 사용자의 선택을 받고, 그에 따른 조건문을 통해 이겼는지, 졌는지, 아니면 비겼는지 등 판별하는 프로그램을 구현했습니다.이벤트 발생 시 alert를 사용하여 결과를 사용자에게 표시가 됩니다.자바스크립트의 기본 개념을 활용하여 간단한 웹 게임을 구현하여 프로그램 능력을 향상 했습니다.

*피라미드 출력 프로그램
수업에서 학습한 반복문에 대한 지식을 활용하여, 사용자로부터 입력받은 정수 값을 '*'로 이루어진 피라미드를 만들어내는 프로그램을 개발하였습니다. 이 프로그램은 반복문을 활용하여 피라미드의 각 층을 구성은 사용자로부터 입력받은 높이에 맞게 출력합니다.사용자가 입력한 정수 값은 피라미드의 높이로 사용되며, 프로그램은 높이를 구성하는 각 층을 출력합니다. 반복문을 통해 각 층의 '*' 을 만들고 출력하여. 반복문의 활용과 사용자 입력 처리에 대한 이해를 깊이 하였습니다.

현재시간 출력 프로그램
Date 객체를 활용하여 현재 서버의 시간을 동적으로 가져오고, 이를 웹 페이지에 실시간으로 표시하는 기능을 구현 하였습니다. Date 객체를 활용하여 현재 시간을 가져와서, 이를 웹 페이지에 동적으로 업데이트합니다. 이를 통해 웹 페이지를 열어두고 있을 때마다 실시간으로 서버의 현재 시간을 확인하는 페이지 입니다.이를 통해 데이터 객체의 활용과 웹 페이지에 동적인 내용을 표시하는 방법에 대하여 공부를 했습니다.
