웹 페이지의 뼈대 구조를 담당하는 HTML


HTML(하이퍼텍스트 마크업 언어)은 웹 페이지의 구조를 정의하는 언어로, 텍스트, 이미지, 링크 등의 요소를 태그를 사용하여 마크업합니다. 브라우저는 이를 해석하여 콘텐츠를 구조화하고 표시합니다. HTML은 웹 페이지의 기본 뼈대를 형성하며, 다른 웹 기술과 함께 사용되어 동적이고 인터랙티브한 경험을 제공합니다.

꾸미고 디자인하는 역할로서 외관을 담당하는 CSS

CSS(캐스케이딩 스타일 시트)는 HTML이나 다른 마크업 언어로 작성된 웹 페이지의 스타일, 레이아웃, 디자인을 정의하는 언어입니다. 선택자(Selector)를 사용하여 HTML 요소를 선택하고, 해당 요소에 적용할 스타일을 속성과 값의 쌍으로 지정합니다. 이를 통해 웹 페이지의 시각적인 표현을 변경하거나 스타일을 지정하여 사용자 경험을 개선할 수 있습니다.

웹 페이지의 뇌를 역할로서 해당하는 Java Script

JavaScript는 웹 페이지를 동적으로 만들고 제어하기 위한 프로그래밍 언어로, 웹 브라우저에서 실행됩니다. HTML과 CSS로 구성된 정적인 웹 페이지를 동적이고 인터랙티브하게 만들기 위해 사용됩니다.
